# Knowledge

Your permanent notes — evergreen ideas, mental models, and synthesized insights.

This is your Zettelkasten: a network of atomic, interconnected notes that grow smarter over time.

## Note Types

- **Fleeting Notes** — Quick thoughts, half-formed ideas, observations
- **Literature Notes** — Summaries of things you've consumed (books, articles, talks)
- **Permanent Notes** — Refined, atomic, self-contained ideas that stand on their own

## Processing Pipeline

```
Fleeting Note → Permanent Note → MOC
    (capture)     (synthesize)      (connect)

Literature Notes live in 04 - Resources/ and feed into Permanent Notes.
```

## Convention

- Each permanent note is **atomic**: one idea per note
- Each permanent note is **self-contained**: makes sense without context
- Each permanent note **links** to at least one other note
- Use `#type/fleeting`, `#type/literature`, `#type/permanent` tags
- Use templates from `90 - Templates/` when creating new notes

## Rules

1. Write notes in your own words — not copied quotes
2. One idea per note — if it covers two ideas, split it
3. Link before filing — always connect to existing notes first
4. Review and refine — permanent notes should improve over time
