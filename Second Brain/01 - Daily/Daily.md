# Daily

Daily notes for journaling, quick reflections, and linking ideas as they emerge.

## Convention

- Each daily note is named `YYYY-MM-DD.md`
- Daily notes are a stream of consciousness — capture moments, meetings, ideas
- Link to other notes liberally using `[[wiki-links]]`
- At the end of each week, review and extract permanent insights into [[05 - Knowledge/Knowledge|Knowledge]]

## Weekly Review

Every Friday or Sunday:
1. Review the week's daily notes
2. Extract reusable ideas → promote to [[05 - Knowledge/Knowledge|Permanent Notes]]
3. Update [[02 - Projects/Projects|Projects]] status
4. Clear any lingering [[00 - Inbox/Inbox|Inbox]] items

## Monthly Review

- Use the Monthly Review template from `90 - Templates/`
- Look for patterns across daily notes
- Update [[03 - Areas/Areas|Areas]] of focus
